# IoT-Machinery-NVH-Monitoring-System# IoT Machinery NVH Monitoring System

## Overview
This project monitors Noise, Vibration and Harshness (NVH) of industrial machinery using Raspberry Pi Pico W.

## Objectives
- Measure vibration
- Measure noise
- Detect abnormal conditions
- Generate alerts
- Upload data to cloud dashboard

## Hardware Used
- Raspberry Pi Pico W
- SW420 Vibration Sensor
- KY038 Sound Sensor
- Buzzer
- LED

## Software Used
- Thonny IDE
- MicroPython
- ThingSpeak
- GitHub

## Applications
- Predictive Maintenance
- Industrial Monitoring
- Machine Health Analysis
